# simple-network-communication

Very simple client/server using TCP.
Server sends message to the client.

# How to use
1. **Compile the code**:
   ```bash
   gcc server.c -o server
   gcc client.c -o client
2. **Run the server**:
   ```bash
   ./server <port>

3. **Run the client**:
   ```bash
   ./client <address> <port>
